# 🐉 Auto-Reroll Monster Hunter Wilds

This PowerShell script automates the **reroll** (reset and reinforcement) process for weapons in **Monster Hunter Wilds**.

Since the game engine (RE Engine) blocks standard software inputs, this script uses the Windows API (`user32.dll`) to simulate **hardware-level** keystrokes (ScanCodes). This allows the game to detect the commands as if they were coming from a real keyboard.

---

## Prerequisites

* **System:** Windows 10 or 11.
* **Permissions:** PowerShell must be run as an **Administrator**.
* **Software:** This script must be used with the **Artian Reroll Tracker** mod enabled. For more details, check the NexusMods page: [https://www.nexusmods.com/monsterhunterwilds/mods/3902?tab=description](https://www.nexusmods.com/monsterhunterwilds/mods/3902?tab=description).

---

## Initial Setup

1. Place the `autoreroll.ps1` script in a dedicated folder.
2. Place the `donnees.json` file in the same directory as the script.
3. Open `donnees.json` with a text editor and enter your current values:
   * Your **Zenny**.
   * Your resource quantities for **Bonuses**.
   * Your **Tarred Devices** quantities for skills.

---

## Using the Auto-Reroll

1. Launch a PowerShell console as **Administrator**.
2. Run the script: `.\autoreroll.ps1`.
3. Choose the desired option (1 for Skills, 2 for Bonuses).
4. Return to the game and place your cursor on the first resource of the relevant menu.
5. Press the **`p`** key on your keyboard.
6. A **3-second** countdown will appear: use this time to stop touching your mouse and keep the game in the foreground (no need to click on the game window again, the script will send inputs to it).
7. The script automatically executes the loops at high speed.
   * *Note:* You might experience PC slowdown during the script's execution. This will confuse the script, and you will need to restart the reroll session.

---

## Technical Operation

* **Skills:** Uses the auto-allocate key `G` to fill the required 1500 points, then confirms with `F`.
* **Bonuses (Phase 1):** Uses `G` to automatically allocate standard resources (Gems, Oricalcite, etc.) until 6000 points are reached, then confirms.
* **Bonuses (Phase 2 - Emblems):** Since the game does not allocate "Battle-forged Emblems" via the `G` key, the script performs a specific sequence (`Z, Z, A, S, F`) to consume them manually.

---

## Limitations and Improvements

The script is currently optimized for speed using the game's native functions. However, it can still be improved:

* **Residual rerolls:** Due to calculation rounding and the game's auto-allocation management, there are usually **5 or 6 rerolls left to do manually** at the end of the process for Bonuses.
* **Resource economy:** The current algorithm consumes resources in whole blocks. A future update could include finer calculations to completely empty the inventory down to the last possible point.

# Mitlu's Reroll Filter (Web Application)

Alongside the PowerShell script, a web application (`index.html` file) is provided to let you easily sort, filter, and analyze the thousands of attempts generated by the Auto-Reroll and tracked by the **Artian Reroll Tracker** mod.

## Web App Features

* **Multi-file Drag & Drop:** Drag and drop multiple JSON files from the mod simultaneously.
* **Dynamic tabs:** Easily navigate between different loaded weapons and reroll sessions.
* **Advanced Filters (Grinding / Bonuses):** Search for precise combinations (e.g., minimum 3 EX bonuses, "All Attack", specific combinations with "AND/OR" conditions).
* **Lottery Filters (Talents/Skills):** Filter your skill attempts by targeting specific "Groups" or "Series".
* **Favorites System (⭐):** Save your best rolls to find them instantly in a dedicated dashboard.

## How to use it?

1. **Retrieve your data:** After completing your in-game reroll sessions with the script and the [Artian Reroll Tracker](https://www.nexusmods.com/monsterhunterwilds/mods/3902) mod enabled, retrieve the `.json` files generated by the mod.
2. **Open the application:** Simply double-click the `index.html` file to open it in your usual web browser (Chrome, Firefox, Edge, etc.). No installation or internet connection is required. Alternatively, you can access it via this link: [https://raven-mitlu.github.io/MonsterHunterWild-AutoReroll-RerollFilter/](https://raven-mitlu.github.io/MonsterHunterWild-AutoReroll-RerollFilter/).
3. **Import the JSONs:** Click the blue **"📁 Load JSON File"** button in the top right corner, or **drag and drop** your `.json` files directly anywhere on the page.
4. **Filter your results:** * If the active tab is a **Bonus (Grinding)** session, configure the dropdown lists to find the perfect combination.
   * If the active tab is a **Skills (Lottery)** session, select the desired group or series.
5. **Save your favorites:** Click the star (☆) on a result card to favorite it. You can find all your sorted favorites by clicking the **"⭐ Favoris"** button in the navigation bar.
6. **Do not refresh the page:** The web application does not create a session; refreshing the page will cause you to lose your favorites and filters.

# Version Francaise
# Auto-Reroll Monster Hunter Wilds

Ce script PowerShell permet d'automatiser le processus de **reroll** (réinitialisation et renforcement) des armes dans **Monster Hunter Wilds**. 

Le moteur du jeu (RE Engine) bloquant les entrées logicielles classiques, ce script utilise l'API Windows (`user32.dll`) pour simuler des pressions de touches au niveau **matériel** (ScanCodes). Cela permet au jeu de détecter les commandes comme si elles provenaient d'un véritable clavier.

---

## Prérequis

* **Système :** Windows 10 ou 11.
* **Droits :** PowerShell doit être lancé en tant qu'**Administrateur**.
* **Logiciel :** Ce script doit être utilisé avec le mod **Artian Reroll Tracker** d'activé. Pour plus de détails, aller voir la page NexusMods : [https://www.nexusmods.com/monsterhunterwilds/mods/3902?tab=description](https://www.nexusmods.com/monsterhunterwilds/mods/3902?tab=description).

---

## Configuration Initiale

1. Placez le script `autoreroll.ps1` dans un dossier dédié.
2. Placez le fichier `donnees.json` dans le même répertoire que le script.
3. Ouvrez `donnees.json` avec un éditeur de texte et renseignez vos valeurs actuelles :
   * Vos **Zenny**.
   * Vos quantités de ressources pour les **Bonus**.
   * Vos quantités d'**Appareils goudronnés** pour les talents.

---

## Utilisation de l'Auto-Reroll

1. Lancez une console PowerShell en **Administrateur**.
2. Exécutez le script : `.\autoreroll.ps1`.
3. Choisissez l'option souhaitée (1 pour les Talents, 2 pour les Bonus).
4. Retournez sur le jeu et placez votre curseur sur la première ressource du menu concerné.
5. Appuyez sur la touche **`p`** de votre clavier.
6. Un décompte de **3 secondes** s'affiche : profitez-en pour ne plus toucher à la souris et laisser le jeu au premier plan (pas besoins cliquer de nouveau sur la fenêtre du jeu, le script lui envvera les inputs).
7. Le script exécute les boucles automatiquement à haute vitesse. 
   * *Note :* Il se peut qu'un ralentissement du PC durant l'exécution du script apparaisse. Cela le déboussolera, il faudra donc recommencer la session de reroll.

---

## Fonctionnement technique

* **Talents :** Utilise la touche d'auto-allocation `G` pour remplir les 1500 points requis, puis valide avec `F`.
* **Bonus (Phase 1) :** Utilise `G` pour allouer automatiquement les ressources standards (Gemmes, Oricalcite, etc.) jusqu'à atteindre 6000 points, puis valide.
* **Bonus (Phase 2 - Emblèmes) :** Comme le jeu n'alloue pas les "Emblèmes forgés dans le combat" via la touche `G`, le script effectue une séquence spécifique (`Z, Z, A, S, F`) pour les consommer manuellement.

---

## Limitations et Améliorations

Le script est actuellement optimisé pour la vitesse en utilisant les fonctions natives du jeu. Cependant, il reste perfectible :

* **Rerolls résiduels :** À cause de l'arrondi des calculs et de la gestion de l'auto-allocation par le jeu, il reste généralement **5 ou 6 rerolls à effectuer à la main** à la fin du processus pour les Bonus.
* **Économie de ressources :** L'algorithme actuel consomme les ressources par blocs entiers. Une future mise à jour pourrait intégrer un calcul plus fin pour vider totalement l'inventaire jusqu'au dernier point possible.

# 🌐 Mitlu's Reroll Filter (Application Web)

En complément du script PowerShell, une application web (fichier `index.html`) est fournie pour vous permettre de trier, filtrer et analyser facilement les milliers d'essais générés par l'Auto-Reroll et traqués par le mod **Artian Reroll Tracker**.

## Fonctionnalités de la Web App

* **Drag & Drop multi-fichiers :** Glissez-déposez plusieurs fichiers JSON issus du mod simultanément.
* **Onglets dynamiques :** Naviguez facilement entre les différentes armes et sessions de reroll chargées.
* **Filtres Avancés (Grinding / Bonus) :** Cherchez des combinaisons précises (ex: 3 bonus EX minimum, "Toute Attaque", combinaisons spécifiques avec des conditions "ET/OU").
* **Filtres Lottery (Talents/Skills) :** Filtrez vos essais de talents en ciblant des "Groupes" ou des "Séries" spécifiques.
* **Système de Favoris (⭐) :** Sauvegardez vos meilleurs jets pour les retrouver instantanément dans un tableau de bord dédié.

## Comment l'utiliser ?

1. **Récupérez vos données :** Après avoir effectué vos sessions de reroll en jeu avec le script et le mod [Artian Reroll Tracker](https://www.nexusmods.com/monsterhunterwilds/mods/3902) activé, récupérez les fichiers `.json` générés par le mod.
2. **Ouvrez l'application :** Double-cliquez simplement sur le fichier `index.html` pour l'ouvrir dans votre navigateur web habituel (Chrome, Firefox, Edge, etc.). Aucune installation ni connexion internet n'est requise. Ou accédez y va ce lien [https://raven-mitlu.github.io/MonsterHunterWild-AutoReroll-RerollFilter/] (https://raven-mitlu.github.io/MonsterHunterWild-AutoReroll-RerollFilter/)
3. **Importez les JSON :** Cliquez sur le bouton bleu **"📁 Load JSON File"** en haut à droite, ou **glissez-déposez** directement vos fichiers `.json` n'importe où sur la page.
4. **Filtrez vos résultats :** * Si l'onglet actif est une session de **Bonus (Grinding)**, configurez les listes déroulantes pour trouver la combinaison parfaite.
   * Si l'onglet actif est une session de **Talents (Lottery)**, sélectionnez le groupe ou la série recherchée.
5. **Sauvegardez vos favoris :** Cliquez sur l'étoile (☆) d'une carte de résultat pour la mettre en favori. Vous pourrez retrouver tous vos favoris classés en cliquant sur le bouton **"⭐ Favoris"** dans la barre de navigation.
6. **Ne pas rafraichir la page :** L'application web ne créé pas de session, rafraichir la page vous fera perdre vos favoris et vos filtres
